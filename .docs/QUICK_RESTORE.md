# Quick Restore Guide

## 🚀 한 줄로 복원하기

Gemini Code Assist에 다음 프롬프트를 입력하세요:

```
다른 PC에서 antigravity를 이용해서 작업한 것을 github의 hodolhodol/newcatalog에 보관했어. 이 PC에 복원해줘.
```

## 📋 AI가 수행할 작업

1. ✅ GitHub에서 저장소 클론
2. ✅ npm 의존성 설치
3. ✅ .env 파일 생성 (NEXTAUTH_SECRET 자동 생성)
4. ✅ Prisma 데이터베이스 초기화
5. ✅ 테스트 데이터 시드
6. ✅ 개발 서버 실행
7. ✅ 문서 확인 및 컨텍스트 로드
